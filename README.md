# LLM Analysis Quiz

This project is part of the TDS LLM Analysis Quiz.

## Description
A Python-based project to analyze LLM responses for quizzes.

## How to Run
1. Install requirements: pip install -r requirements.txt
2. Run main program: python main.py

## Docker
To run in Docker: docker build -t llm-quiz .
docker run -p 5000:5000 llm-quiz
